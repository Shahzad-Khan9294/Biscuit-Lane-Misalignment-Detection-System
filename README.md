# Biscuit-Lane-Misalignment-Detection-System
AI-Powered Conveyor Lane Integrity & Misalignment Detection System

This project is a real-time computer vision quality control system designed for high-speed industrial production lines — specifically multi-lane conveyor environments (e.g., biscuit manufacturing lines).

➤  What This System Does?
It continuously monitors a 24-lane conveyor system and:
- Tracks product flow lane-by-lane
- Learns and stabilizes lane boundaries dynamically
- Detects strip misalignment in real time
- Identifies cross-lane drift and boundary violations
- Triggers instant alerts (audio + visual)
- Captures and stores evidence frames
- Generates an annotated monitoring output video

## In high-throughput manufacturing environments, even small strip misalignments can lead to:
- Product collisions
- Packaging errors
- Machine jams
- Material wastage
- Downtime
- Rework costs
This solution acts as an AI-based production guardrail, continuously verifying that product streams remain geometrically aligned and operationally compliant.

➤ What Makes This Powerful?

✅ Slope-Aware Lane Preservation

✅ Per-Lane Intelligent Misalignment Detection

✅ Temporal Noise Filtering

✅ Extreme Boundary Monitoring

✅ Automated Evidence Capture
